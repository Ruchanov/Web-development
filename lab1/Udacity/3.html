In June 2011, Google redirected all traffic from google.kz to google.com in response to a demand issued by the Kazakh Ministry of Communications and Information that requires all .kz domain names to operate on physical servers within the borders of Kazakhstan.

A second country code top-level domain, ("qaz") for Kazakhstan is now active. It is used with web addresses using Cyrillic letters. It was launched in March 2012, when the first site, a test site (тест.қаз, "test.qaz"), was activated.
